# 小程序开发模板

## 安装
```sh
yarn install
```

## 调试
```sh
yarn dev
```


## 打包
```sh
yarn build
```

## 目录结构说明

config:taro 配置  
src:开发目录  
--components:组件（全局组件，全局方法）  
----taro-ui-styles:组件样式  
--pages:页面  
----_template:页面模版  
--static:放置一些静态文件  
--app.scss:样式入口  
--app.tsx:js入口  
--config.ts:配置文件  
--globalStore.ts:全局数据
--index.html:html5项目的入口  
tools:帮助项目开发的脚本目录
